BRchan X

BRchan X é um userscript com diversas modificações para melhorar a visualização e usabilidade do brchan. Foi criado com base no 4chan X (aeosynth, mayhem).

Última versão

Clique aqui para instalar a última versão do script.

Compatibilidade

O script foi testado nos browsers Firefox 15 (Pale Moon) e Chrome 21. A compatibilidade com outras versões/outros browsers não é garantida, use por sua própria conta e risco.
